# Titanic ML Workflow

## Objective
- Kaggle Titanic 데이터를 이용해
- 데이터 전처리 → EDA → 베이스라인 모델 → 검증
- 데이터 공모전 기본 워크플로우를 정리

## Dataset
- Kaggle Titanic (train.csv)

## Workflow
1. Data loading & cleaning
2. Exploratory Data Analysis
3. LightGBM baseline
4. Cross Validation
5. Feature Engineering

## Key Takeaways
- 결측치 처리 전략 정리
- 범주형 변수 인코딩 방식 비교
- CV 기반 성능 평가 중요성
